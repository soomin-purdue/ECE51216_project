# Local Benchmark Set

This repository includes a small local benchmark set so the solver can run.

Included inputs:

- `sat/example-1.cnf`
- `sat/uf20-01.cnf`
- `sat/uf20-04.cnf`
- `unsat/unsat.cnf`
- `unsat/unsat1.cnf`
- `unsat/unsat2.cnf`
- `unsat/unsat3.cnf`

These files are sufficient for basic validation, intermediate report screenshots, and early
benchmarking before larger benchmark collections are added.
